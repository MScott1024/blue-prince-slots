The program will start 3 seconds after you choose which machine you're on. Press 'escape' or use the default pyautogui failsafe and move your mouse to a corner of your screen. Only works for 1080p screens, if yours is not you can change the values at the top of the main python file to match your screen. (coordinates are (x,y) where (0,0) is the top left corner of your screen) If you run in to issues do tell.
