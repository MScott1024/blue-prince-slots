TO INSTALL: go to the release to the left, download the zip file called Blue Prince Slot Machine Bot, and extract it. If you want to move the .exe file just right click it, and create a shortcut
Press space to start and escape to exit.
Only works for 1920x1200 screens, if yours is not that and you have python installed, you can change the region values at the top of the main python file to match your screen. If you don't have python installed I could possibally do it for you.
I like using "GT autoclicker" on the microsoft store to find the pixel coordinates, though it's only accurite if you're display is at 100%.
(coordinates are (x,y) where (0,0) is the top left corner of your screen)
