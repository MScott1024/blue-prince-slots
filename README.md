TO INSTALL: download the zip file called Blue Prince Slot Machine Bot, then extract it. If you want to move the .exe file just right click it, and create a shortcut
Press space to start and escape to exit.
Only works for 1920x1200 screens, if yours is not that you can change the values at the top of the main python file to match your screen.
I like using "GT autoclicker" on the microsoft store to find the pixel coordinates.
(coordinates are (x,y) where (0,0) is the top left corner of your screen)
If you run in to issues do tell.
